üöÄ Installation Guide

Download the Desktop App

Click the "Download for Windows" button above to get the FindYourTab desktop application. Extract the ZIP file to a folder of your choice.

Install Browser Extension

Install the FindYourTab extension from the Chrome Web Store (or Firefox Add-ons). This allows the desktop app to communicate with your browser tabs.

Run the Desktop App

Launch FindYourTab.exe from the extracted folder. The app will start in your system tray and begin monitoring your browser tabs.

Start Using FindYourTab

Press Ctrl+Alt+F anywhere to open the tab finder. Type to search, then click any tab to switch to it instantly!

üí° How to Use

  • Open Tab Finder: Press Ctrl+Alt+F from anywhere
  • Search Tabs: Start typing to filter tabs by title, URL, or domain
  • Switch Tabs: Click on any tab or use arrow keys + Enter
  • Close Tab Finder: Press Escape or click outside
  • System Tray: Right-click the tray icon for settings and options

üîß System Requirements
