Quick Setup

1

Download App

Download and run FindYourTab.exe. Windows may show a security warning - click "More info" then "Run anyway".

2

Install Extension

Add the browser extension from Chrome Web Store or load the unpacked extension from the project files.

3

Start Using

Press Ctrl+Alt+F to open the tab finder. Click any tab to switch to it instantly!
